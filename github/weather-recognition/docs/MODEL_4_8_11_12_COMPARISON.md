## 6. 矛盾现象解释

### 6.1 CrossEntropyLoss vs FocalLoss 数据增强效果相反

| 配置    | CrossEntropyLoss | FocalLoss  |
| ----- | ---------------- | ---------- |
| 有数据增强 | 81.07%           | 77.34%     |
| 无数据增强 | 76.13%           | **83.18%** |

### 6.2 原因分析

Focal Loss 的 alpha 权重基于 **无数据增强** 的 Model 1 每类准确率计算：

```python
focal_loss_per_class_acc = {
    "cloudy":  0.4965,  # 无增强环境下的准确率
    "haze":    0.7831,
    "rainy":   0.6405,
    ...
}
alpha = 1.0 / (per_class_acc + eps)  # 权重与无增强分布绑定
```

数据增强改变了样本分布，导致：

1. 权重高估/低估了某些类的难度
2. 数据增强引入的"伪困难样本"被过度聚焦
3. 模型学习方向偏离

### 6.3 验证方法

用 Model 10（有数据增强）的每类准确率重新计算 alpha，再训练 Focal Loss + 数据增强的模型，如果准确率提升则证明假设正确。

## 7. 训练稳定性分析

| 模型       | 最佳 epoch | 最终 train\_acc | 最终 val\_acc | 过拟合差距  | 收敛稳定性   |
| -------- | -------- | ------------- | ----------- | ------ | ------- |
| Model 4  | 78       | 88.46%        | 70.94%      | 17.52% | 中等      |
| Model 8  | -        | -             | -           | -      | 好       |
| Model 11 | 19       | \~97%+        | \~77%       | \~20%  | 好（早停触发） |
| Model 12 | 26       | 99.89%        | 82.32%      | 17.57% | 好       |

**分析**：

- Model 12 过拟合差距最小（17.57%）
- CosineAnnealing 有效减少过拟合
- 无数据增强让模型学到更真实的特征

## 8. 结论与建议

### 8.1 模型推荐

| 场景       | 推荐模型         | 准确率        | 理由                                       |
| -------- | ------------ | ---------- | ---------------------------------------- |
| **最佳综合** | **Model 12** | **83.18%** | Focal Loss + 分层 + CosineAnnealing（无数据增强） |
| 生产推荐     | Model 10     | 81.07%     | CrossEntropyLoss + 数据增强 + 分层             |

### 8.2 Focal Loss 使用建议

- **无数据增强时**：Focal Loss γ=1.0 能有效提升困难类
- **有数据增强时**：CrossEntropyLoss 表现更好
- **如需结合两者**：必须重新计算 alpha 权重

### 8.3 最优配置总结

| Loss 函数      | 数据增强 | 分层划分 | CosineAnnealing | 准确率        |
| ------------ | ---- | ---- | --------------- | ---------- |
| FocalLoss    | -    | Y    | Y               | **83.18%** |
| CrossEntropy | Y    | Y    | -               | 81.07%     |
