## 1. 评估指标说明

### 1.1 Precision（查准率）

**定义**：预测为某类的样本中，实际为该类的比例。

$$
P = \frac{TP}{TP + FP}
$$

- TP（True Positive）：预测正确且实际为该类
- FP（False Positive）：预测为该类但实际不是

**意义**：你预测"这张是晴天"，有多大把握是对的**

### 1.2 Recall（查全率）

**定义**：实际为某类的样本中，被正确预测的比例。

$$
R = \frac{TP}{TP + FN}
$$

- FN（False Negative）：实际为该类但预测为其他类

**意义**：所有晴天图片，你找出来了多少**

### 1.3 F1 Score

**定义**：Precision 和 Recall 的调和平均数，平衡两者。

$$
F1 = \frac{2 \times P \times R}{P + R}
$$

**意义**：查准和查全的综合衡量，F1 高表示两者都较好**

### 1.4 Macro F1

**定义**：先计算每个类别的 F1，再取算术平均。

$$
\text{Macro F1} = \frac{1}{8} \sum_{i=1}^{8} F1_i
$$

**意义**：对所有类别一视同仁，不受样本数量影响。适合关注每个类别的均衡表现**

### 1.5 Weighted F1（加权 F1）

**定义**：每个类别的 F1 按该类的样本数加权平均。

$$
\text{Weighted F1} = \frac{\sum_{i=1}^{8} (F1_i \times N_i)}{\sum_{i=1}^{8} N_i}
$$

- $N_i$ 为第 $i$ 类的样本数

**意义**：受样本数量影响，大类别的表现对总体影响更大**

### 1.6 总准确率（Accuracy）

**定义**：预测正确的样本占总样本的比例。

$$
\text{Acc} = \frac{TP_1 + TP_2 + ... + TP_8}{\text{总样本数}}
$$

**意义**：模型整体正确率，但无法反映各类的细分表现**

## 6. 关键发现

### 6.1 γ 值的有效性边界

| γ 范围 | 效果 | 说明 |
|--------|------|------|
| γ > 1.5 | 过度聚焦 | 简单类衰减严重，haze 下降 11%+ |
| γ = 1.0~1.5 | **最佳区间** | 平衡困难类提升与简单类保持 |
| γ = 0.5 | **极差** | 衰减太轻，接近 CE，shine 无法识别 |
| γ = 0 | 等同 CE | 无 Focal Loss 效果 |

### 6.1 γ=1.0 综合表现最优

Model 4（γ=1.0）在：
- 总准确率：74.29%（最高）
- Macro F1：0.7717（最高）
- Weighted F1：0.7444（最高）
- cloudy F1：0.5732（提升 +13.4% vs 基线）
- rainy F1：0.7219（提升 +5.8% vs 基线）
- shine F1：0.8767（提升 +8.0% vs 基线）

### 6.2 γ 值与困难类别表现的关系

| γ 值 | 特点 | 困难类表现 |
|------|------|---------|
| 2.0 | 激进衰减 | sunny +5.5%，cloudy +1.5% |
| 1.5 | 温和衰减 | sunny +12.7%，cloudy -5.7% |
| 1.0 | 轻度聚焦 | cloudy +13.4%，rainy +5.8% |
| 0.5 以下 | 几乎无效 | 接近普通 CE |

**结论**：γ=1.0 达到了最佳的平衡——既对困难类别有明显提升，又不会过度衰减简单类别。

### 6.3 简单类别在不同 γ 下的稳定性

| 类别 | γ=2.0 | γ=1.5 | γ=1.0 | γ=1.75 | γ=1.25 |
|------|-------|-------|-------|--------|--------|
| thunder | 0.9408 | 0.9268 | 0.9298 | 0.9263 | 0.9262 |
| sunrise | 0.9259 | 0.8952 | 0.9320 | 0.9143 | 0.9000 |
| shine | 0.8298 | 0.8211 | 0.8767 | 0.8696 | 0.9014 |

γ=1.0 对 thunder/sunrise/shine 的衰减最小，简单类别保持得最好。

## 7. 学习率调度对比分析（Model 4 vs Model 8）

| 对比项 | Model 4（无调度） | Model 8（CosineAnnealing） | 变化 |
|--------|-------------------|---------------------------|------|
| γ | 1.0 | 1.0 | 相同 |
| 总准确率 | 74.29% | **77.52%** | **+3.23%** |
| Macro F1 | 0.7717 | **0.7931** | **+0.0214** |
| Weighted F1 | 0.7444 | **0.7760** | **+0.0316** |
| cloudy F1 | 0.5732 | 0.5479 | -0.0253 |
| haze F1 | 0.6667 | **0.7855** | **+0.1188** |
| rainy F1 | 0.7219 | 0.6937 | -0.0282 |
| shine F1 | 0.8767 | **0.8675** | -0.0092 |
| snow F1 | 0.7896 | **0.8571** | **+0.0675** |
| sunny F1 | 0.6834 | **0.7341** | **+0.0507** |
| sunrise F1 | 0.9320 | **0.9242** | -0.0078 |
| thunder F1 | 0.9298 | **0.9346** | **+0.0048** |

### 7.1 CosineAnnealing 带来的提升

- **Macro F1 提升 +2.1%**：从 0.7717 → 0.7931
- **haze 大幅提升 +11.9%**：0.6667 → 0.7855（最显著）
- **snow 提升 +6.8%**：0.7896 → 0.8571
- **sunny 提升 +5.1%**：0.6834 → 0.7341
- **thunder 略微提升 +0.5%**：0.9298 → 0.9346

### 7.2 代价与权衡

- **cloudy 轻微下降**：0.5732 → 0.5479（-2.5%），但仍在可接受范围
- **rainy 轻微下降**：0.7219 → 0.6937（-2.8%）

### 7.3 结论

CosineAnnealing 学习率调度使模型整体表现大幅提升，**所有简单类（haze、snow、sunny、thunder）均获得明显提升**，而困难类（cloudy、rainy）虽有轻微下降，但整体 Macro F1 仍有 +2.1% 的显著提升。这说明 CosineAnnealing 帮助模型更好地收敛到更优的全局解。
